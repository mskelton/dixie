[![npm version](https://img.shields.io/npm/v/npm-template)](https://www.npmjs.com/package/npm-template)

## Installation

### npm

```sh
npm install npm-template
```

### Yarn

```sh
yarn add npm-template
```

### pnpm

```sh
pnpm add npm-template
```

### bun

```sh
bun add npm-template
```
